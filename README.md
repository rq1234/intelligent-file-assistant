# File Organiser AI

An intelligent file organization system that automatically categorizes and moves files based on their content and context.

## Features

- Automatic file watching and detection
- Content-based file classification
- AI-powered decision making
- Undo functionality for file operations
- Local storage for state management

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## Configuration

Edit `config/settings.yaml` to customize the behavior of the file organiser.
